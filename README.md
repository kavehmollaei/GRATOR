## Installation & Runing
``` 
$ cd GRATOR 
$ pip3 install -r requirements.txt
$ python3 grator.py 
$ python3 grator.py start
$ python3 grator.py stop
``` 
## Guide 

To see duplicate words and display their numbers, we have the following command

* -f For Find and Show 
* -r For Remove 
* -c For Path file
